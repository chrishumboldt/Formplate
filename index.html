Rocket Form Module

Getting Started

Install via NPM.

npm install rocket-form

Note that this module has a dependency Rocket Tools (21kb) which will automatically be installed as well.

Start by including the necessary files.

<head>
    <link href="node_modules/rocket-form/css/form.min.css" rel="stylesheet" type="text/css">
</head>
<body>
    /* Your content goes here */
    <script src="node_modules/rocket-tools/js/tools.min.js"></script>
    <script src="node_modules/rocket-form/js/form.min.js"></script>
</body>

Javascript Call

<script>
Rocket.form({
    targets: '.form-el',
    colour: 'red',
    style: 'raised'
});
// With Shifting Label
Rocket.form({
    targets: '.form-el-2',
    colour: 'red',
    label: 'shift',
    style: 'raised'
});
</script>
